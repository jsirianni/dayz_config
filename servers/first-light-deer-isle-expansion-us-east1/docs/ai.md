# DayZ Expansion AI Configuration

Reviewed: 2026-05-31

This server uses DayZ Expansion AI with custom patrol definitions for Deer Isle. The important files are split between the mission folder and the profile folder:

- `empty.deerisle/expansion/settings/AIPatrolSettings.json` controls patrol groups, spawn coordinates, faction, behavior, loadout name, respawn/despawn rules, and load balancing category.
- `empty.deerisle/expansion/settings/AILocationSettings.json` defines named roaming locations used by local roaming AI.
- `profiles/ExpansionMod/Settings/AISettings.json` defines global AI combat and movement behavior.
- `profiles/ExpansionMod/Loadouts/*.json` defines the equipment trees referenced by patrol `Loadout` values.
- `profiles/ExpansionMod/Logs/ExpLog_2026-02-22_12-27-02.log` shows Expansion tried to generate default Deer Isle patrols but found no default spawn points. The custom JSON appears to be the effective patrol source.

## High-Level Summary

- Expansion AI patrols are enabled.
- There are 130 patrol definitions.
- All patrols use `LoadBalancingCategory: "Patrol"`.
- The `Patrol` load balancing category allows a maximum of 5 active patrols for 0-255 players.
- The `Global` load balancing category also applies and decreases the total active patrol budget as player count rises.
- All patrols have `Chance: 0.99`.
- All patrols have `RespawnTime: -2.0`, so they inherit the file-level respawn time of 600 seconds.
- File-level spawn/despawn distances are:
  - `MinDistRadius: 400.0`
  - `MaxDistRadius: 1000.0`
  - `DespawnRadius: 1100.0`
  - `DespawnTime: 600.0`
  - `RespawnTime: 600.0`
- Every patrol has `UseRandomWaypointAsStartPoint: 0`, so the first waypoint is the spawn point.
- Every patrol has an empty `ObjectClassName`, so these are normal coordinate patrols, not object-bound patrols.
- Every patrol has an empty `Units` list, so the unit identities are generated by Expansion from the patrol count, faction, and loadout.
- Every patrol uses `Speed: "RANDOM_NONSTATIC"` and `UnderThreatSpeed: "SPRINT"`.

## Spawn And Roaming Model

Expansion patrol waypoints are world coordinates in `[x, y, z]` form. For this server, `y` is usually `0.0`, letting the game resolve terrain height.

For `ROAMING_LOCAL` patrols, only the starting waypoint matters. Expansion uses that start point to pick the nearest configured roaming location from `AILocationSettings.json`, then the AI roam around buildings/points in that local area.

For `LOOP` patrols, the AI follow the listed waypoints in a loop.

For `HALT` patrols, the AI hold position unless combat or threat behavior causes them to react.

## Loadouts

Expansion loadout JSON is a nested item tree. Clothing and gear are usually in `InventoryAttachments` with slot names such as `Body`, `Legs`, `Feet`, `Back`, `Vest`, `Headgear`, `Gloves`, `Hips`, `Melee`, `Shoulder`, and `Hands`. Random firearm choices are usually in the root `Sets` array as `ClassName: "WEAPON"` entries.

### `WestLoadout`

Used by West and Mercenary patrols.

Gear theme:

- USMC desert/woodland jacket and pants.
- Combat boots.
- Coyote backpack.
- Plate carrier or high-capacity vest.
- Ballistic helmet or Mich2001 helmet.
- Tactical gloves and military belt.

Firearms:

- Scout with `Mag_Scout_5Rnd`, possible ACOG/M68/Reflex/Starlight optics.
- FAMAS with `Mag_FAMAS_25Rnd`.
- M4A1 with CMAG/STANAG magazine options and M4 furniture/optic options.
- Aug with `Mag_Aug_30Rnd`.
- AugShort with `Mag_Aug_30Rnd` and optic options.
- M16A2 with STANAG magazine options.

Patrol-level values are mostly `-1`, which means they inherit the global AI settings.

## Things To Watch

- Mercenary patrols now use `WestLoadout`; keep future patrol loadout names aligned with files in `profiles/ExpansionMod/Loadouts`.
- Do not assume all 130 patrols are active at once. The `Patrol` category cap is 5.
- Do not confuse `AILocationSettings.json` with direct spawn points. It names local roaming areas; actual patrol spawn points are in `AIPatrolSettings.json`.
- `ROAMING_LOCAL` patrols with a single waypoint can still move widely inside the nearest local roaming area.
- If adding more patrols, update load balancing if the goal is to actually increase active AI presence.
